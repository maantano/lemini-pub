사용자의 법률 질문에서 관련 법령과 검색 키워드를 추출하라.

## 추출 규칙

1. 질문의 핵심 법률 쟁점을 파악하라.
2. 해당 쟁점을 규율하는 한국 법령명을 반드시 포함하라.
3. 법령에서 사용하는 정식 법률 용어를 포함하라.
4. 질문에 사용된 일상 용어의 법률적 동의어를 포함하라.
5. 핵심 키워드와 동의어/유의어를 함께 추출하라.
6. 너무 일반적인 단어는 단독으로 쓰지 말고, 구체적 행위와 결합된 복합 명사로 추출하라.
7. 오타가 있으면 원본과 교정본을 모두 포함하라.
8. 조사(은/는/이/가), 어미, 감탄사는 제외하라.

반드시 JSON 배열로만 응답하라.
